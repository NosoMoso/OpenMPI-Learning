
Для работы с OpenMPI в Visual Studio необходимо скачать пакет HPC Pack 2012 MS-MPI

После установки необходимо подключить HPC Pack 2012 MS-MPI к своему проекту:

 1. Указываем в заголовке библиотеку #include <mpi.h>
 2. В свойствах проекта C\C++ -> Доп. каталоги подключаемых файлов -> Указываем "C:\Program Files\Microsoft HPC Pack 2012\Inc"
 3. В свойствах проекта Компановщик -> Доп. каталоги библиотек -> Указываем "C:\Program Files\Microsoft HPC Pack 2012\Lib\i386"
 4. В свойствах проекта Компановщик -> Ввод -> Доп. зависимости -> Указываем "msmpi.lib"

Для запуска скомпилированного exe файла необходимо в консоле ввести команду:

 mpiexec -n 10 OpenMPI.exe

где -n 10 - задаваемое число узлов
